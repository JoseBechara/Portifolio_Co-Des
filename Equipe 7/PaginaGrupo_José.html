Projetos físicos e mecânicos

Melhoria para carroças de catadores

Neste projeto os alunos de Natureza do Design entrevistaram vários carroceiros para saber as dificuldades diárias enfretadas por eles, e assim criarem soluções a serem implementadas em suas carroças para melhorar as suas condições de trabalho.

Girafa do Job Rotation

Neste projeto os alunos de Natureza do Design construiram várias girafas feitas a base de MDF, utilizando os equipamentos do FabLab, como uma forma de aprender qual a função e como manusear cada uma das máquinas.

Estação Metereológica

Neste projeto os alunos de Instrumentação e Medição construiram uma estação meteorológica utilizando os sensores calibrados por eles mesmos durantes as aulas e o Arduino, criaram e soldaram a placa de circuitos com os sensores para acoplar ao Arduino, bem como programá-lo.

Projeto Final de Acionamentos Elétricos

Neste projeto os alunos de Acionamentos Elétricos construiram um dispositivo capaz de acionar um motor através de PWM (Modulação Por Largura de Pulso).

Projeto Final de Modelagem e Controle

Neste projeto os alunos de Modelagem e Controle fizeram um robô capaz de andar se equilibrando automaticamente. Nesta materia os alunos devem ajustar os parâmetros de uma malha de controle proporcional, PD e PID para que o controlador correspondente siga determinadas especificações funcionais.
